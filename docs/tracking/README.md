# Project Tracking

This folder contains project tracking documents for the RunRoute app.

## Contents

- **[backlog.md](backlog.md)** - Feature backlog and planned enhancements
- **[issues.md](issues.md)** - Known issues and bugs

## Current Focus

See the [Project Plan](../plan/projectplan.md) for overall project status and roadmap.

## How to Use

1. **Adding new work items**: Add to `backlog.md` with priority level
2. **Reporting bugs**: Add to `issues.md` with reproduction steps
3. **Starting work**: Move items from backlog to current sprint section

## Priority Levels

- **P0**: Critical - blocking release
- **P1**: High - should be addressed soon
- **P2**: Medium - nice to have
- **P3**: Low - future consideration
